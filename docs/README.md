## 💡 文档使用技巧

### 搜索内容

在大多数 Markdown 阅读器中：
- 使用 `Ctrl+F` (Windows/Linux) 或 `Cmd+F` (Mac) 搜索关键词
- 利用目录快速跳转到相关章节

### 代码示例

所有代码示例都可以：
- 直接复制到项目中使用
- 根据实际情况调整参数
- 作为参考模板编写自己的代码

### 交叉引用

文档之间相互引用：
- 从一个文档可以跳转到另一个文档
- 相关主题会有链接提示
- 建议按推荐顺序阅读

## 📝 文档规范

### 文件格式
- 所有文档使用 Markdown (.md) 格式
- 编码: UTF-8
- 换行符: LF (Unix) 或 CRLF (Windows)

### 命名规范
- 文件名使用大写和下划线
- 例如: `QUICK_START.md`, `API_DOCUMENTATION.md`

### 内容结构
- 清晰的标题层级
- 适当的代码示例
- 表格展示对比信息
- Emoji 增强可读性
